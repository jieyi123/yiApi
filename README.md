# API开放平台

## 项目介绍

这是一个提供API接口供开发者调用的平台，基于Spring Boot后端+React前端的全栈微服务项目。
管理员可以接入并发布接口、统计分析各接口调用情况；用户可以注册登泉并开通接口调用权限、浏览
接口、在线调试，还能使用客户端SDK轻松在代码中调用接口。

在线体验地址：http://api.originai.icu/
